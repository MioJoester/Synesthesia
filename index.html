About Synesthesia

Synesthesia is an experimental, open-source audio visualization project that transforms sound into motion. It uses real-time frequency analysis to drive 3D particle systems, creating an immersive visual response to audio input.

How it works

  • • Web Audio API for FFT analysis
  • • Three.js for GPU-accelerated rendering
  • • Audio-driven particle displacement

Technology

  • • HTML5
  • • JavaScript (ES6)
  • • Three.js
  • • Tailwind CSS
  • • particles.js
